## 3. ApiEndpoint Component

Document API endpoints with interactive examples:

import ApiEndpoint from '@site/src/components/ApiEndpoint';

<ApiEndpoint
  method="POST"
  path="/api/v1/execute"
  description="Execute a trading order with risk validation"

## Best Practices

### When to use each component:

- **Tabs**: Multi-language code examples, different configuration options
- **CodeBlock**: Standalone code with title and line numbers
- **ApiEndpoint**: REST API documentation with request/response examples
- **Mermaid**: Architecture diagrams, sequence flows, state machines

### Styling Tips:

- Keep diagrams simple and focused
- Use consistent colors (Indigo for backend, Cyan for analytics, Green for UI)
- Add titles to code blocks for clarity
- Include both request and response examples in API docs
