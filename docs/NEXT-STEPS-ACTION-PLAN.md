#### 5. Optimize Sidebar Configuration (~3 hours)

**Owner**: DocsOps  
**Effort**: 3 hours  
**Impact**: Medium (maintenance reduction)

**Goal**: Reduce manual sidebar configuration from 355 lines to ~100 lines by using autogeneration + frontmatter.

**Current** (manual):
```javascript
items: [
  { type: 'doc', id: 'tools/overview' },
  {
    type: 'category',
    label: 'Infrastructure',
    items: [{ type: 'autogenerated', dirName: 'tools/infrastructure' }],
  },
  {
    type: 'category',
    label: 'Monitoring',
    items: [{ type: 'autogenerated', dirName: 'tools/monitoring' }],
  },
  // ... 7 more manual categories
],
```

**Improved** (autogenerated):
```javascript
items: [{
  type: 'category',
  label: 'Tools',
  link: { type: 'doc', id: 'tools/overview' },
  items: [{ type: 'autogenerated', dirName: 'tools' }],
}],
```

**Then control ordering via frontmatter**:
```markdown
---
title: Infrastructure Tools
sidebar_position: 1
---
```

**Benefits**:
- Automatic navigation updates when files added
- Less configuration to maintain
- Consistent ordering via frontmatter
