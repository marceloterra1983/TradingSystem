## Cleanup Policy

Files are moved to archive when:
1. ✅ Issue is completely resolved
2. ✅ Documentation superseded by newer versions
3. ✅ Session-specific notes no longer relevant
4. ✅ Information consolidated into canonical docs

Files are removed when:
1. ❌ Completely obsolete with no historical value
2. ❌ Duplicate information already archived
3. ❌ Temporary debugging notes
