# Style Guide

## Microcopy

- Prefer concise, action-oriented labels (for example, “Place Order” instead of “Execute an order now”).
- Use sentence case for buttons and navigation.

## Dates & Numbers

- Display dates in `YYYY-MM-DD` for trading logs.
- Use locale-aware formatting for monetary values but keep precision to two decimals unless specified.
