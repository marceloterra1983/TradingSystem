# Deployment Checklist Overview

Esta página resume os pontos obrigatórios antes de disparar um deploy. Para a lista detalhada, consulte o controle completo em [`/governance/controls/pre-deploy-checklist`](/governance/controls/pre-deploy-checklist).

## Itens essenciais

- ✅ Variáveis de ambiente críticas validadas (`bash scripts/validation/validate-env.sh`)
- ✅ Docker Compose validado (`docker compose ... config`)
- ✅ Portas e redes conferidas (`bash scripts/validation/check-port-conflicts.sh`)
- ✅ Targets de proxy revisados nas stacks frontend (`VITE_*_PROXY_TARGET`)
- ✅ Consultas SQL filtrando `__checkpoint__`
- ✅ Status `received,queued` considerados nos consumidores
