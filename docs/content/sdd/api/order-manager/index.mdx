## Resumo

O Order Manager é o serviço responsável por aplicar verificações de risco, enviar ordens via ProfitDLL e sincronizar o ciclo de vida das execuções. Esta página centraliza os artefatos que descrevem a API e os padrões operacionais do serviço.

## Documentos principais

- [Especificação da API](./v1/spec) — contrato REST completo da versão 1.
- [Guidelines de implementação](./v1/guidelines) — convenções, padrões e processos recomendados.
- [Changelog](./v1/changelog) — histórico de decisões e evolução da API.

## Contexto adicional

- `backend/api/order-manager/` — caminho planejado para implementação do serviço.
- Integração ProfitDLL — consulte `docs/tools/profitdll/overview` para requisitos da stack.
- Para runbooks de operações, utilize a seção `docs/apps/order-manager/`.
