## Overview

TradingSystem uses autonomous agents to automate market analysis, risk management, and signal orchestration. The current implementation is the **Agno Agents system**, a Python 3.12 microservice using the Agno framework.

**Agent Systems**:

1. **Agno Agents** (Active) - Multi-agent trading system with 3 specialized agents
   - MarketAnalysisAgent: Generate trading signals from market data
   - RiskManagementAgent: Validate signals against risk policies
   - SignalOrchestratorAgent: Coordinate agent workflows

**Future Agent Systems** (Planned):

- Order Execution Agents: Automate order placement and management
- Portfolio Optimization Agents: Multi-asset portfolio balancing
- Backtesting Agents: Historical strategy validation

## Agent Catalog

### Agno Agents System

**Technology**: Python 3.12 + FastAPI + Agno framework + OpenAI

**Port**: 8200

**Status**: Active (MVP)

**Agents**:

1. **MarketAnalysisAgent**
   - Responsibility: Analyze TP Capital signals and Workspace ideas
   - Output: BUY/SELL/HOLD recommendations with confidence scores
   - Integration: TP Capital API (3200), Workspace API (3100)

2. **RiskManagementAgent**
   - Responsibility: Validate signals against risk limits
   - Checks: Daily loss limit, position size limit, trading hours
   - Output: RiskAssessment with approval decision

3. **SignalOrchestratorAgent**
   - Responsibility: Coordinate MarketAnalysisAgent and RiskManagementAgent
   - Actions: ANALYZE, VALIDATE, ORCHESTRATE
   - Output: Aggregated signals and assessments

## Integration Guides

### Dashboard Integration

**Proxy Configuration**: Dashboard proxies `/api/agents` to `http://localhost:8200`

**Usage Example**:

```typescript
const response = await fetch('/api/agents/analyze', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    symbols: ['PETR4', 'VALE3'],
    include_tp_capital: true
  })
});
const data = await response.json();
```

### CLI Integration (Planned)

**Custom Command**: `/agents analyze PETR4`

```bash
curl -X POST http://localhost:8200/api/v1/agents/analyze \
  -H "Content-Type: application/json" \
  -d '{"symbols": ["$1"], "include_tp_capital": true}' \
  | jq '.signals'
```

## Troubleshooting

| Symptom | Possible Cause | Recommended Action |
|---------|----------------|--------------------|
| 503 on `/api/v1/agents/analyze` | TP Capital API indisponível | Check logs, wait for circuit HALF_OPEN, verify `/health?detailed=true` |
| Metrics zeroed | Prometheus not configured | Ensure scrape job `agno-agents` in Prometheus config |
| LLM not responding | `OPENAI_API_KEY` invalid | Validate credentials, check quota, review error logs |
| High latency | External APIs slow | Monitor `dependency_status`, adjust timeout, implement caching |
