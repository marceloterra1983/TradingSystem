## API Standards

All TradingSystem APIs follow these standards:

### Documentation
- âœ… OpenAPI 3.1 specification
- âœ… Interactive Redocusaurus documentation
- âœ… Code examples in multiple languages
- âœ… Comprehensive error documentation

### Rate Limiting
- **Default:** 300 requests per 60-second window
- **Headers:** `RateLimit-Limit`, `RateLimit-Remaining`, `RateLimit-Reset`
- **Response:** `429 Too Many Requests` when exceeded

### Authentication
- **Current:** API key (development)
- **Planned:** JWT tokens, OAuth2, mTLS (production)

### Observability
- âœ… Prometheus metrics at `/metrics`
- âœ… Health checks at `/health`
- âœ… Structured logging (JSON)
- âœ… Request tracing (correlation IDs)

### Versioning
- **Strategy:** URL-based versioning (`/api/v1/...`)
- **Deprecation:** 6-month notice period
- **Support:** N and N-1 versions maintained

## Integration Examples

### JavaScript/TypeScript

```typescript
import axios from 'axios';

// Documentation API
const docsApi = axios.create({
  baseURL: 'http://localhost:3400',
});

const systems = await docsApi.get('/api/v1/systems');

// Workspace API
const workspaceApi = axios.create({
  baseURL: 'http://localhost:3200',
});

const items = await workspaceApi.get('/api/items');
```

### Python

```python
import requests

# Documentation API
docs_response = requests.get('http://localhost:3400/api/v1/systems')
systems = docs_response.json()

# Workspace API
workspace_response = requests.get('http://localhost:3200/api/items')
items = workspace_response.json()
```

## API Governance

### Adding New APIs

1. **Create OpenAPI Spec**
   ```bash
   # Place spec in canonical location
   touch docs/static/specs/new-api.openapi.yaml
   ```

1. **Configure Redocusaurus**
   Edit `docs/docusaurus.config.js`:
   ```javascript
   specs: [
     // ... existing specs
     {
       id: 'new-api',
       spec: 'static/specs/new-api.openapi.yaml',
       route: '/api/new-api',
     },
   ]
   ```

1. **Create Documentation**
   ```bash
   touch docs/content/api/new-api.mdx
   ```

1. **Update Navbar**
   Add link to navbar dropdown in `docusaurus.config.js`

See `docs/static/specs/README.md` for detailed instructions.
