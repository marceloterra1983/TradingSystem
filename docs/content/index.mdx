## Contributor workflow

1. **Sync automation**: `npm --prefix docs run docs:auto` then `npm --prefix docs run docs:validate-generated`.
2. **Develop locally**: `npm --prefix docs run docs:dev -- --port 3400 --host 0.0.0.0` (matches dashboard proxy).
3. **Run the full check suite**: `npm --prefix docs run docs:check` before opening a PR.
4. **Validate technical references**: `bash docs/scripts/validate-technical-references.sh --strict`.
5. **Publish**: merge to main → CI builds static assets → documentation container consumes latest build.

## Automation at a glance

| Command | Description |
|---------|-------------|
| `npm --prefix docs run docs:auto` | Sync generated tables (ports, design tokens, registries). |
| `npm --prefix docs run docs:validate-generated` | Ensure generator markers and timestamps are consistent. |
| `npm --prefix docs run docs:check` | Full pipeline (auto → validate → lint → type-check → tests → build). |
| `npm --prefix docs run docs:links` | Build + external link validation. |
| `bash docs/scripts/validate-technical-references.sh --strict` | Catches stale paths/ports (expects 3400/3401). |
