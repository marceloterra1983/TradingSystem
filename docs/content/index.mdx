## Contributor workflow

1. **Sync automation**: `npm --prefix docs run docs:auto` then `npm --prefix docs run docs:validate-generated`.
2. **Develop locally**: `npm --prefix docs run docs:dev -- --port 3400 --host 0.0.0.0` (matches dashboard proxy).
3. **Run the full check suite**: `npm --prefix docs run docs:check` before opening a PR.
4. **Validate technical references**: `bash docs/scripts/validate-technical-references.sh --strict`.
5. **Publish**: merge to main â†’ CI builds static assets â†’ documentation container consumes latest build.
