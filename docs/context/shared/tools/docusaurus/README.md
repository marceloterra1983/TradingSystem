### Starting Docusaurus

```bash
cd docs/docusaurus
npm install
npm run start -- --port 3004 --host 0.0.0.0
```

### Building for Production

```bash
npm run build
npm run serve
```

### Clearing Cache

```bash
npm run clear
```
