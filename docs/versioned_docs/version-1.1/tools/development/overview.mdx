### 3. Firecrawl (`tools/firecrawl/`)

**Open-source web scraping tool for converting websites to markdown.**

**Structure:**

```

```

**Purpose:**
- Web scraping (extract content from websites)
- Markdown conversion (HTML â†’ clean Markdown)
- Document ingestion (feed scraped content to LlamaIndex RAG)
- Content archival (snapshot websites as Markdown)

**Usage:**

```bash
# Start Firecrawl Proxy (Port 3600)
docker compose -f tools/compose/docker-compose.firecrawl.yml up -d

# Scrape URL
curl -X POST http://localhost:3600/api/scrape \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}'

# Response (Markdown)
{
  "markdown": "# Example Domain\n\nThis domain is for use in...",
  "metadata": {
    "title": "Example Domain",
    "url": "https://example.com",
    "timestamp": "2025-10-27T12:00:00Z"
  }
}
```

**Integration with LlamaIndex:**

```python
# Scrape and ingest into RAG
import requests
from llama_index import Document

# Scrape URL
response = requests.post(
    "http://localhost:3600/api/scrape",
    json={"url": "https://docs.example.com"}
)
markdown = response.json()["markdown"]

# Create document
doc = Document(text=markdown, metadata={"source": "https://docs.example.com"})

# Ingest into LlamaIndex
index.insert(doc)
```

**Features:**
- JavaScript rendering (Playwright-based)
- Anti-bot bypass (stealth mode)
- Rate limiting and retry logic
- Proxy support
