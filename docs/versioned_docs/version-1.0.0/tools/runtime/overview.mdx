## üöÄ Quick Start

### Check Installed Versions

```bash
# Node.js
node --version  # v20.x or later
npm --version   # v10.x or later

# Python
python3 --version  # 3.11 or later
pip3 --version

# .NET
dotnet --version  # 8.0 or later

# Docker
docker --version
docker compose version
```

### Install Dependencies

```bash
# Node.js dependencies (frontend/backend)
npm install

# Python dependencies (AI tools)
pip3 install -r requirements.txt

# .NET dependencies (trading services - planned)
dotnet restore
```

### Run Services

```bash
# Node.js service
npm run dev

# Python service (with venv)
source venv/bin/activate
python main.py

# .NET service
dotnet run

# Docker service
docker compose up -d
```
