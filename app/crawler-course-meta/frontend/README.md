# Crawler Course Meta – Frontend

Dashboard Next.js 15 + Tailwind para monitorar jobs do backend.

## Scripts

```bash
npm install
npm run dev
```

Defina `NEXT_PUBLIC_API_URL` apontando para a API (`http://localhost:8080`).

### Rotas

- `/` – overview e KPIs
- `/jobs` – listagem
- `/jobs/new` – wizard JSON
- `/metrics` – métricas Prometheus
- `/artifacts` – viewer de downloads
- `/settings` – checklist de segurança
